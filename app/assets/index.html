Material Icon Generator

Drop SVG icon here

SVG must contain a single path which is your icon.

What is this?

This is a work in progress tool for generating simple, material style inspired, icons with a long shadow cast to the bottom right.

Why?

Making icons is fun. Creating the material shadow is not so much fun. This tool helps me to quickly create launcher icons for Android apps.
